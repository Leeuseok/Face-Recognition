## 💡 주요 기능
- **얼굴 감지**
  - OpenCV 라이브러리를 호출하여 웹캠으로부터 영상 가져옴
  
- **아두이노 제어**
  - 시리얼 통신을 사용하여 데이터를 주고 받아 아두이노 제어
  
- **아두이노 반응**
  - 명령에따라 LED를 켜거나 모터를 작동시킴
